<product_requirements>
The project is to build ReMindMe, an AI-powered SaaS relationship manager. The mission is to help users maintain meaningful connections by managing contacts, setting smart reminders, and generating personalized messages.

**Core MVP Features:**
1.  **User Authentication:** Secure signup/login via email.
2.  **Contact Management:** Manual add/edit and import from CSV. Profiles include fields like birthday, relationship, and notes.
3.  **Smart Reminders:** System for birthdays, anniversaries, and follow-ups, with a calendar view.
4.  **AI Message Generation:** Integration with Gemini to draft personalized messages with selectable tones (Friendly, Professional, etc.).
5.  **Dashboard & Analytics:** An overview of upcoming events, stale contacts, and key metrics.

**Design & Performance Enhancements (Phase 2):**
- **Visuals:** A clean, minimalist, and emotionally warm UI using a modern design system, soft shadows, rounded corners, and animations (Framer Motion).
- **Performance:** Optimize loading times using code-splitting, lazy loading, image optimization, and skeleton loaders for a perceived performance boost.
- **UX:** Improve user flow with microinteractions and a polished, responsive layout.

The goal was to build a launch-ready MVP that is both highly functional and delivers a polished, performant user experience.
</product_requirements>
